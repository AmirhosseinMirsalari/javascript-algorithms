# چالش: پیدا کردن عدد گم‌شده

## دستورالعمل‌ها

یک تابع به نام `findMissingNumber` بنویسید که یک آرایه از اعداد یکتا از 1 تا n (شامل) را دریافت می‌کند، جایی که یک عدد گم‌شده است. باید عدد گم‌شده را برگرداند.

### مثال‌ها

```js
findMissingNumber([1, 2, 3, 4, 6, 7, 8, 9, 10]); // 5
findMissingNumber([10, 8, 6, 7, 5, 4, 2, 3, 1]); // 9
findMissingNumber([10, 5, 1, 2, 4, 6, 8, 3, 9]); // 7
```

### محدودیت‌ها

- اگر یک آرایه خالی داده شود، باید `1` را برگرداند
- اگر هیچ چیزی داده نشود، باید `undefined` را برگرداند

### راهنماها

- مجموع اعداد از 1 تا n (شامل) را محاسبه کنید. فرمول برای این کار `n * (n + 1) / 2` است. `n` طول آرایه به علاوه 1 است.
- مجموع اعداد در آرایه را محاسبه کنید.
- مجموع اعداد در آرایه را از مجموع اعداد از 1 تا n (شامل) کم کنید.
- می‌توانید از حلقه for یا متد reduce برای محاسبه مجموع اعداد در آرایه استفاده کنید. ما در بخش بعد به روش‌های مثل reduce تمرکز خواهیم کرد، اما همچنان هر دو روش را به شما نشان خواهم داد.

```js
function findMissingNumber(arr) {
  if (!arr || arr.length === 0) {
    return undefined;
  }

  const n = arr.length + 1;
  const expectedSum = (n * (n + 1)) / 2;

  let actualSum = 0;
  for (let i = 0; i < arr.length; i++) {
    actualSum += arr[i];
  }

  return expectedSum - actualSum;
}

// استفاده از reduce
function findMissingNumber(arr) {
  if (!arr || arr.length === 0) {
    return undefined;
  }

  const n = arr.length + 1;
  const expectedSum = (n * (n + 1)) / 2;

  const actualSum = arr.reduce((sum, num) => sum + num, 0);

  return expectedSum - actualSum;
}
```

### توضیح

تابع `findMissingNumber` یک آرایه از اعداد یکتا از 1 تا n (شامل) را دریافت می‌کند، جایی که یک عدد گم‌شده است، و عدد گم‌شده را برمی‌گرداند.

ابتدا مجموع اعداد از 1 تا n (شامل) را با استفاده از فرمول `n * (n + 1) / 2` محاسبه می‌کنیم. این فرمول به نام فرمول گوس است. این مقدار را در یک متغیر به نام `expectedSum` ذخیره می‌کنیم.

برای روشن‌تر شدن بیایید مثالی را بررسی کنیم. اگر یک آرایه از اعداد از 1 تا 10 داشته باشیم، مجموع این اعداد 55 است. می‌توانیم این مقدار را با استفاده از فرمول گوس محاسبه کنیم. `10 * (10 + 1) / 2 = 55`. همچنین می‌توانیم این مقدار را با استفاده از یک حلقه for یا متد reduce محاسبه کنیم.

سپس مجموع اعداد در آرایه را محاسبه می‌کنیم و این مقدار را در یک متغیر به نام `actualSum` ذخیره می‌کنیم.

سپس تفاوت مجموع انتظاری و مجموع واقعی را برمی‌گردانیم. این مقدار عدد گم‌شده خواهد بود.

در مجدد می‌تو

انید از یک حلقه for یا متد reduce برای محاسبه مجموع اعداد در آرایه استفاده کنید.

### موارد آزمون

```js
test("پیدا کردن عدد گم‌شده", () => {
  expect(findMissingNumber([1, 2, 3, 5])).toBe(4);
  expect(findMissingNumber([10, 8, 6, 7, 5, 4, 2, 3, 1])).toBe(9);
  expect(findMissingNumber([1, 3, 4, 5, 6])).toBe(2);
});
```
